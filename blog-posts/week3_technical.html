JavaScript

By Shreya Patel

July 3, 2014


You’re probably wondering what JavaScript is? Here’s what it come down to:

JavaScript is an object orientated programming language that is built into all major web browsers. It is used to create interactive features on webpages making websites more dynamic.


There’s an increasing popularity of JavaScript usage and here’s why:

  • JavaScript is used on the front end to create interactive elements on websites
  • It works greatly alongside HTML and CSS
  • JavaScript comes installed on every modern web browser, so it’s useable right way
  • Since its client-side, there is less load on the server, making it speedy
  • JavaScript is simple and quick to learn, making it a popular language even amongst high school computer courses
  • Tons of job growth and high pay for those who know JavaScript

What about the backend?

  • JavaScript not only runs in the frontend, but also in the backend
  • You can build servers in JS using Node.js
  • AJAX created the ability to use JS in conjunction with data
  • JavaScript Object Notation allows transferring of information from a database to website
  • Frameworks like, Ember.js, Angular.js, and Backbone.js, are all powering the creation of interactive web applications
  • JavaScript is a great fit for mobile development platforms forms because
    • It has the ability to meet the demands of rapid development
    • Maximizes the user experience

While programming languages are constantly improving, there are downsides such as the following,

  • Since JavaScript code runs on the user’s computer it can be accessible by others who may potentially exploit the user’s system
  • JavaScript is rendered differently by different browsers. Client-side scripts can vary compared to server-side scripts, where the output is always the same. You want make sure to test your JS script on various browsers for consistency.
  • Users may have client-side code block, which eliminates viewers.
