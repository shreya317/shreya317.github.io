Ruby: Arrays and Hashes

By Shreya Patel

August 7th, 2014

Arrays

Definition:

  • An array is a list of items in order
  • An array can contain any type of legitimate Ruby object
  • Every slot in the list acts like a variable
    • You can see what object a particular slot points to
    • You can make it point to different object
    • Example:
      • [ ] #empty array
      • [1] #array holding a single number
      • [“Ruby”, “DBC”] #array holding two strings
      • name = “Shreya” #assignment
      • [2.1, name, [“yes”, 1] ] #array holding three objects; float, string, and array.

Integer & Zero Indexed

  • To help us find a particular object in an array, each slot is given an index number
  • The first slot in the array is always indexed number zero, the second is indexed number 1, third is indexed number 2, and so forth.

Accessing Values

  • You can store arrays into variable names and access the values by calling the index number
  • Example: languages = [ “Ruby”, “JavaScript”, “HTML”, “CSS” ]
    • puts languages[0] #=> Ruby
    • puts languages [1] #=> JavaScript
    • puts languages [2] #=> HTML
    • puts languages[3] #=> CSS
    • puts languages #=>Ruby JavaScript HTML CSS

Creating Arrays

  • You can create arrays multiple ways, the simplest way is to assign an empty set of square brackets to a variable (array literal)
    • a = [ ]
  • You can also create an array using the “new” method of the Array class
    • a = Array.new

Adding and Removing Items

  • To add an item to an array at the end, use the push method
    • pets = ["cat", "dog"]
    • pets.push("hamster")
    • #=> pets = ["cat", "dog", "hamster"]
  • To add an item to the beginning of the array, use the unshift method.
    • pets.unshift(“turtle”)
    • #=> pets = [“turtle”, “cat”, “dog”, “hamster”]
  • To remove the last item of an array, use the pop method. This will remove and return the value of that item
    • pets.pop
    • #=> hamster
  • To remove the first item of an array, use the shift method
    • pets.shift
    • #=> turtle

Using other bulit-in methods

  • To get the length of an array, use the length method
    • pets = [“turtle”, “cat”, “dog”, “hamster”]
    • pets.length
    • #=> 4
  • To find out what the first item of the array is, use the first method
    • pet.first
    • #=> turtle
  • To find out what the last item of the array is, use the last method
    • pet.last
    • #=> hamster

Iterating

  • You can iterate through an array using the each method. This allows us to execute a block of code to each item within the array
    • pets = [“turtle”, “cat”, “dog”, “hamster”]
    • pets.each { |animal| puts “I love my “ + animal + “.”}
    • # The code between curly braces is what we will do to each item in the array
    • # |animal| is the variable that each will use to point to the objects in the array
    • #=> I love my turtle.
    • #=> I love my cat.
    • #=> I love my dog.
    • #=> I love my cat.

What are they good for?

  • Arrays are good to keep track of list of strings, to store references to items you are working with, and to iterate through groups of objects in a sequence.

Hashes

Definition

  • A hash is a collection of key-value pairs.
    • Example: { “Jack” => “turtle, “Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”}
  • They are similar to arrays, but instead of integer indexes, you use the keys to index.
  • You can use any Ruby object as key or value, even an array.
    • Example: {[1, “jan”] => “January"}
  • Hash values don’t have to be unique, you can assign the same value two or more keys
  • You can't have duplicate keys

Accessing Values

  • You can assign a variable name to a hash and access its values by calling the hash key, like so
    • pet_type = { “Jack” => “turtle, “Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”}
    • pet_type["Jack"] #=> turtle
    • pet_type["Thomas"] #=> cat
    • pet_type["Scruffy"] #=> dog
    • pet_type["Tiny"] #=> hamster

Creating Hahes

  • You can create hashes multiple ways, the simplest way is to assign an empty set of curly braces to a variable (hash literal)
    • pet_type = { }
  • You can also create a hash using the new class method
    • pet_type = Hash.new

Adding & Removing

  • To add a new key,value pair, use the [ ] = method
    • pet_type = { “Jack” => “turtle, “Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”}
    • pet_type[“Fluffy”] = “bunny”
    • #=> { “Jack” => “turtle, “Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”, “Fluffy” => “bunny”}
  • To remove key, value pair, use delete method followed by the key
    • pet_type = { “Jack” => “turtle, “Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”}
    • pet_type.delete(Jack)
    • #=> {“Thomas” => cat, “Scruffy” => dog, “Tiny” => “hamster”, “Fluffy” => “bunny”}

Iterating

  • You can iterate over a hash using the each method
    • pet_type = {"Jack" => "turtle", "Thomas" => "cat", "Scruffy" => "dog", "Tiny" => "hamster"}
    • pet_type.each {|key, value| puts "#{key} is my #{value}."}
    • #=> Jack is my turtle.
    • Thomas is my cat/
    • Scruffy is my dog.
    • Tiny is my hamster.

What are they good for?

  • It’s good to use hashes to keep track of things that don’t fit in an ordered list. Hashes let you perform lookup operations based on keys.

When to use what...

If I wanted a list of all the names in my phonebook then I could use an array to store the list of names.

  • phonebook = ["Jack", "Alex", "Sam", "Brian", "Kelly"]

I could alphabetize the list by using the sort method,

  • phonebook.sort
  • #=> [“Alex”, “Brian”, “Jack”, “Kelly”, “Sam”]

If I wanted all the entries in the phonebook with name and corresponding phone number, I would use a hash to store the entries.

  • phonebook = {“Jack” => 1111111, “Alex” => 2222222, “Sam” => 3333333, “Brian” => 4444444, “Kelly” => 5555555

If I wanted Sam’s phone number, I would get this information by calling the key.

  • phonebook["Sam"]
  • #=> 3333333
